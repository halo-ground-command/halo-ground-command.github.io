# Halo: Ground Command — Force Builder

A lightweight, browser-based force builder for **Halo: Ground Command**.  
Build Battle Groups, track points/BR, validate required slots and caps, and export a roster.

## Use it online
Force Builder: https://halo-ground-command.github.io/

## Features
- Build forces by **Faction** and **Battle Group**
- Tracks **Points** and **BR**
- Validates **Required** slots and **Optional** caps
- Unit **upgrades** support (including required “choose 1” groups)
- **Copy roster** output for sharing
- **Save/Load JSON** list files
- **Print / Export to PDF** (via browser print dialog)

## Run locally
1. Download the `.html`
2. Open the `.html` file in your browser
